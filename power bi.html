Interactive Power BI Sales Dashboard

Description

Created a dynamic Power BI dashboard to analyze and visualize sales performance. The dashboard integrates data from multiple sources to provide actionable insights, helping drive sales strategy and decision-making.

Key Features

Sales Metrics: Displays total sales, revenue growth, and sales targets.

Geographical and Trend Analysis: Visualizes regional sales performance and time-based trends.

Product Insights: Highlights top-performing products and categories.

Interactive Filters: Allows users to filter data by date, region, product, and sales rep.

Technologies Used

Power BI, CRM systems, Databases, Excel, DAX, Power Query
